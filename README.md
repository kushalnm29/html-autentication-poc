# HTML Authentication POC

This project is a basic Proof of Concept (POC) for an authentication flow using only HTML.

## Pages Included
- login.html
- register.html
- forgot-password.html
- reset-password.html
- dashboard.html

## Features
- Navigation using anchor tags
- Simple authentication flow simulation
- No CSS or JavaScript used (HTML only)

## Flow
Login → Dashboard  
Register → Login  
Forgot Password → Reset Password → Login
